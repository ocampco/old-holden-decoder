### Commands

```sh
# Install dependencies
$ nvm use
$ pnpm i

# Run UI
# http://localhost:5173/
$ pnpm run dev

# Run Vitest tests
$ pnpm run test

# Run eslint and prettier
$ pnpm run lint

# Auto fix using prettier
$ pnpm run lint:fix
```
